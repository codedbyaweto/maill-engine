# MailEngine

MailEngine is a modern email campaign management platform that allows users to create, manage, and track email campaigns with real-time analytics, templates, and engagement insights. It is built for speed, scalability, and a smooth user experience across all devices.

---

## Features

### Campaign Management
- Create, edit, and delete email campaigns
- Draft, schedule, and send campaigns
- Track campaign status (sent, scheduled, failed, etc.)
- Paginated campaign list with filtering and search

### Analytics Dashboard
- Deliverability trends (bounce rate, complaint rate, delivery rate)
- Engagement scoring per recipient
- List health scoring and recommendations
- Visual charts for performance insights

### Template System
- Drag & drop email builder
- HTML editor support
- Upload custom HTML templates
- Template categorization and search
- Paginated template library

### Team Management
- Invite and manage team members
- Role-based access control
- View and manage user permissions

### ⚡ Performance & UX
- Skeleton loading states across all pages
- Responsive UI for mobile and desktop
- Fast navigation with Next.js routing
- Optimistic UI patterns

---

## Tech Stack

- **Frontend:** Next.js (App Router), React, TypeScript
- **Styling:** Tailwind CSS
- **State/Data:** RTK Query
- **UI Components:** Custom component system + shadcn-style primitives
- **Charts:** Recharts
- **Icons:** Lucide React
